# Uso-de-IA-para-An-lise-Financeira---Explore-o-Poder-do-NotebookLM
Projeto prático de uso de IA na análise financeira, explorando o NotebookLM para estudar dados, gerar insights e testar prompts. Inclui curadoria de fontes, resumos estruturados, glossário e engenharia de prompts para apoiar decisões e aprendizagem ativa em finanças.
